# Crypto Portfolio Optimization with Efficient Frontier

This personal project performs **cryptocurrency portfolio optimization** using **Modern Portfolio Theory (MPT)** and the **PyPortfolioOpt** library. It constructs optimal portfolios under realistic constraints and visualizes the **efficient frontier**, including **minimum volatility** and **maximum Sharpe ratio** portfolios.

---

## Features

- Uses **historical cryptocurrency price data**
- Computes:
  - Expected returns (mean historical returns)
  - Risk model using Ledoitâ€“Wolf covariance shrinkage
- Optimizes portfolios under **custom allocation constraints**
- Generates:
  - Minimum volatility portfolio
  - Maximum Sharpe ratio portfolio
- Visualizes:
  - Efficient frontier
  - Individual assets
  - Optimized portfolio intersections
